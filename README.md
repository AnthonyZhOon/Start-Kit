# Start-Kit

## Compile the start-kit

Using `compile.sh`:
```shell
./compile.sh
```

Using cmake: 
```shell
mkdir build
cd build
cmake ../ -DCMAKE_BUILD_TYPE=Release
make -j
```

## Run the start kit

Runing startkit using
command: 
```
./lifelong --inputFile the_input_file_name -o output_file_location
```
more info on help
